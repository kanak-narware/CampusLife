# **App Name**: CampusLife Navigator

## Core Features:

- Attendance Tracker: Track attendance for each subject, mark presence/absence, and view subject-specific attendance statistics.
- Timetable Manager: Organize and manage class schedules with a clear, visual timetable.
- Assignment Feed: Receive, view, and track assignments posted by a community admin; filter and sort assignments by due date, subject, and priority.
- AI-Powered Study Tool Recommendation: Recommends personalized study tools based on current assignments and class schedule. It acts as a tool to help decide what type of study aid will be best.
- Dashboard Overview: Provides a daily overview of classes, assignments, and attendance status.

## Style Guidelines:

- Primary color: Light blue (#ADD8E6), evoking calmness and clarity, fitting for an academic environment.
- Background color: Very light gray (#F0F0F0), offering a neutral and clean backdrop to improve readability and focus.
- Accent color: Muted blue (#77B5FE), providing a subtle contrast for interactive elements without overwhelming the user.
- Body and headline font: 'PT Sans', a humanist sans-serif, providing a modern yet warm feel, suitable for both headlines and body text.
- Use simple, clear icons for navigation and features, ensuring ease of use and quick recognition.
- Employ a clean, grid-based layout to present information in a structured and easily digestible manner.
- Use subtle animations for transitions and interactions to provide feedback without distracting from the core functionality.
